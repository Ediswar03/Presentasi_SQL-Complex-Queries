# 📖 MANUAL PENGGUNAAN APLIKASI PRESENTASI

Panduan ini menjelaskan fitur teknis aplikasi presentasi "Mastering SQL Complex Queries" dan cara mengoperasikannya dengan lancar.

---

## 🎮 Kontrol Navigasi
Aplikasi ini dirancang untuk fleksibilitas tinggi selama presentasi:

| Aksi | Tombol Keyboard | Gerakan Mouse/Sentuh |
| :--- | :--- | :--- |
| **Slide Selanjutnya** | `Panah Kanan`, `Spasi` | Klik area slide atau Swipe Kiri |
| **Slide Sebelumnya** | `Panah Kiri` | Swipe Kanan |
| **Membuka Menu** | `M` | Klik ikon Grid di Navbar |
| **Layar Penuh** | `F` | Klik ikon Maximize di Navbar |
| **Tutup Menu** | `Esc` | Klik di luar area menu |

---

## 🛠️ Fitur-Fitur Unggulan

### 1. Slide Overview (Quick Jump)
Tekan **'M'** untuk membuka grid navigasi. Anda bisa melompat ke slide mana pun tanpa harus melewati slide lainnya satu per satu. Sangat berguna saat sesi tanya jawab.

### 2. Mode Terang & Gelap
Tersedia tombol toggle tema di pojok kanan atas.
*   **Mode Gelap (Default):** Cocok untuk proyektor di ruangan redup, terlihat sangat modern.
*   **Mode Terang:** Gunakan jika ruangan sangat terang agar teks lebih mudah dibaca.

### 3. Copy-to-Clipboard
Pada setiap slide yang menampilkan kode SQL, terdapat tombol **Salin** di pojok kanan atas blok kode. Saat diklik, ikon akan berubah menjadi centang hijau menandakan kode berhasil disalin.

### 4. Interactive Optimization Comparison
Pada slide **⚡ Optimasi Performa**, terdapat dua tombol: `Slow Query` dan `Optimized`.
*   Klik untuk menunjukkan perbandingan query yang membebani server vs query yang efisien.
*   Gunakan ini untuk menjelaskan konsep **SARGable** (Search Argumentable) kepada audiens.

### 5. Bar Progres Interaktif
Bar garis di bagian bawah bukan sekadar hiasan. Anda bisa **mengklik bagian mana pun pada bar tersebut** untuk berpindah ke posisi slide yang diinginkan secara instan.

---

## 📑 Cara Cetak ke PDF
1. Klik ikon **Printer** di navbar.
2. Jendela cetak browser akan terbuka.
3. Ubah tujuan (Destination) menjadi **"Save as PDF"**.
4. Pastikan opsi "Background Graphics" dicentang agar desain glassmorphism tetap muncul.
